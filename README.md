# 纪念日追踪 (Day Counter)

记录和追踪重要日子的网页工具，显示每个事件距离今天还有多少天或已过去多少天。

## 功能

- **添加事件**：设置名称、日期和表情符号
- **倒计时/正计时**：自动计算距离目标日期的天数（未到/已过/今天）
- **事件排序**：按日期自动排序，最近的排在最前
- **已过事件标记**：过去的日期自动灰显并降低透明度
- **删除事件**：点击 × 按钮移除不需要的事件
- **数据持久化**：使用 localStorage 保存，关闭浏览器不丢失
- **中英文切换**：界面支持中文/英文

## 使用方法

1. 用浏览器打开 `index.html`
2. 在底部输入框填写事件名称、选择日期、可选表情符号
3. 点击"添加"按钮
4. 卡片会显示距离该日期还有多少天
5. 点击卡片右侧 × 可删除事件

## 技术栈

纯 HTML/CSS/JavaScript，使用 localStorage 实现数据持久化。
